
# SQLite
Is an employing file-system base databasees
for the use of SQL without a separate server installation

SQLite is ideal for working with SQK offline and without a dedicated server proceses

## create or open database:
sqlite3 mydatabase.db


## Execute SQL commands:
CREATE TABLE users (id INT, naem VARCHAR(255));
INSERT INTO users VALUES (1, 'Alice');
SELECT * FROM users;

## Exit
.quit or .exit
